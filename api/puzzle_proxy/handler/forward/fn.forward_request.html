Forward a request to the upstream server using hyper client.

H7: Applies connect timeout and total request timeout. H-3: When resolved_addrs is provided, uses the pre-resolved IP addresses to connect instead of re-resolving DNS (prevents DNS rebinding TOCTOU). M17: Streams the request body instead of buffering entirely in memory.
