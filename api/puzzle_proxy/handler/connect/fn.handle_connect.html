Handle a CONNECT request (HTTPS tunneling via TLS passthrough).

C9: Establishes a TCP connection to the target host, sends “200 Connection Established” back to the client, then relays bytes bidirectionally between client and upstream. H7: Applies connect timeout and total tunnel timeout.
