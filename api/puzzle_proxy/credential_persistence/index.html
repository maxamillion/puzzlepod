§3.4 G12: Credential mapping persistence for restart recovery.

Persists phantom token → credential spec mappings to credential_mappings.json in the branch state directory. This file contains phantom tokens and backend references — never real credential values.

File format:

{
  "version": 1,
  "proxy_port": 18443,
  "mappings": [
    { "phantom_token": "pt_abc123_...", "credential_name": "api-key", "domains": ["api.example.com"] }
  ]
}
