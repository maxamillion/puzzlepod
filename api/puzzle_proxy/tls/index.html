TLS MITM proxy support for agent network inspection.

C4: Provides per-agent CA certificate generation and on-the-fly leaf certificate issuance for TLS interception. This allows the proxy to decrypt, inspect, and journal HTTPS POST/PUT/DELETE/PATCH requests while passing through GET requests.

Each agent branch gets its own ephemeral CA (valid 1 day). The CA cert PEM is injected into the agent’s trust store so it accepts the proxy’s leaf certs. Leaf certs are generated on-the-fly for each target domain (valid 1 hour).
