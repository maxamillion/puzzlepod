Expand description

Encrypted credential store backed by AES-256-GCM.

Fields§

§credentials: HashMap<String, StoredCredential>

Credentials keyed by name.

§store_path: PathBuf

Path to the encrypted store file on disk.

§encryption_key: Zeroizing<[u8; 32]>

D-M3: AES-256-GCM encryption key derived via HKDF from the signing key. Wrapped in Zeroizing for structural zeroization on drop.

Create a new credential store.

Derives the encryption key from signing_key via HKDF-SHA256. If the store file already exists on disk, its contents are decrypted and loaded.
