Module credentials 

Source
Expand description

Credential store and phantom token manager for the agent proxy.

Provides secure credential storage with AES-256-GCM encryption at rest, domain-scoped credential injection, and phantom token issuance so that agents never see real secrets — only surrogate tokens that the proxy resolves at request time.

PXH4: All credential operations are auditable via D-Bus signals.
