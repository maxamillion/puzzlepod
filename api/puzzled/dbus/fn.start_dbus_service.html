Expand description

Start the D-Bus service. Returns the connection and the initialized flag; the caller must set initialized to true after all subsystems are ready, and keep the event loop alive (e.g., via ctrl_c wait).

M23: The initialized flag is NOT set inside this function. The caller sets it after D-Bus + WAL recovery + policy load + seccomp hardening are all complete, ensuring agents cannot register prematurely.
