Module budget

Expand description

Adaptive budget engine for agent resource allocation.

Agents start in the Restricted tier. After a configurable number of clean commits, they escalate to Standard, then Extended. Any policy violation drops them one tier.

Tier changes adjust cgroup resource limits (memory, CPU) and storage quotas dynamically. This implements the “trust through behavior” principle — agents earn more resources by demonstrating safe behavior.
