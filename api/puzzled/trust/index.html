Graduated Trust with Behavioral Learning (PRD Section 4.1).

Tracks per-UID trust scores that drive privilege escalation / de-escalation. Scores are updated by audit events (commit approved, policy violation, etc.) and behavioral baseline anomaly detection. State is persisted to disk as JSON so that trust survives daemon restarts.

§Thread Safety

TrustManager is not internally synchronized. Callers must wrap it in Arc<Mutex<TrustManager>> or equivalent when accessed from multiple async tasks (e.g., concurrent D-Bus handlers for the same UID).
